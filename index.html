Configurazione

Imposta il collegamento a Google Sheets e i parametri per l'invio automatico delle e-mail ai clienti.

Come collegare Google Sheets (operazione una tantum):
1. Apri il tuo Google Sheet → Estensioni → Apps Script
2. Incolla il codice dal commento README_SETUP in fondo a questo file HTML
3. Clicca Distribuisci → Nuova distribuzione → Web App
   Esegui come: me stesso · Accesso: chiunque
4. Autorizza e copia qui l'URL generato.

Logo: posiziona logo.png accanto al file HTML e scrivi logo.png.
